These adult group ROIs (9mm, spherical) were defined from a sample of n=20 adults (see Richardson et al., 2018 for details).
They have been resampled to the MNI template used by fMRIPrep.
Theory of Mind: RTPJ, LTPJ, DMPFC, VMPFY, MMPFC, PC
Pain matrix: RS2, LS2, Rinsula, Linsula, RMFG, LMFG, AMCC


The FS_ROI-lut.tsv lists the current FreeSurfer subcortical segmentations that can be used as ROIs in the pipeline.
See the github wiki page for more information about how to use these labels.
